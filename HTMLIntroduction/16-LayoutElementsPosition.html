Position

Position özelliği bir eleman için kullanılan konumlandırma yönteminin türünü belirtir.

Daha sonra öğeler top, bottom, left ve right özellikleri kullanılarak konumlandırılabilir.

  • Static
  • Relative
  • Fixed
  • Absolute
  • Sticky

Static

Html elementleri default olarak statictir. Static elemanlar top, bottom, left, right özelliklerinden etkilenmezler.

Relative

Html elementleri default olarak statictir. Static elemanlar top, bottom, left, right özelliklerinden etkilenmezler.

Fixed

Html elementleri görünüm alanına göre konumlandırılır. Ekrana sabitlenir.

Absolute

Element en yakın konumlandırılmış atasına göre konumlandırılır. ancak absolute olarak işaretlenmiş elementin konumlandırılmış bir atası yoksa belge gövdesi kullanılır ve sayfa kaydırmasıyla birlikte hareket eder. Mutlak elementler normal akıştan çıkarılır bu elemenlerin üst üste gelmesine neden olabilir.

Sticky

Kaydırma konumanı bağlı olarak relative ve fixed arasında geçiş yapar.
