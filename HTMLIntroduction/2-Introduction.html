1-HTML MEDİR? (HYPER TEXT MARKUP LANGUAGE) HTML, web sayfalarının yapısını oluşturmak için kullanılan bir işaretleme dilidir. HTML, web sayfalarının içeriğini ve yapısını tanımlamak için kullanılır. HTML, web sayfalarının metin, resim, video, ses, bağlantılar ve diğer içeriklerini düzenlemek için kullanılır. HTML YAPISI: Html, etiket (tag) adı verilen özel yapılar kullanarak sayfalar oluşturur. Bu etiketler başlangıc(acilis) ve bitiş(kapanış) etiketlerinden oluşur. Her etiketin belirli bir amacı vardır ve tarayıcıya ne yapması gerektiğini söyler. HTML Etiket ve Element Kavramı: Html çeşitli TAGler kullanarak sayfa yapısını oluşturur. (html, head, body, h1, title, p gibi) Başlangıç ile bitiş tagi arasında kalan tüm bölüme ELEMENT denir. EMMET? Emmet, HTML ve CSS yazarken hız kazanmanızı sağlayan bir eklentidir. Emmet, kısayollar kullanarak HTML ve CSS kodlarını hızlı bir şekilde yazmanıza olanak sağlar.
