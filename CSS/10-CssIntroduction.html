CSS-CASCADING STYLE SHEET

CSS teknik olarak bir programlama dili değil bir stil tanımlama dilidir. Css, html gibi işaretleme dilleri ile kullanılarak, web sayfalarının görünümünü ve tasarımını belirlemek için kullanılır.

Html dokumana 3 farklı şekilde CSS uygulanabilir.

  • İnternal CSS

    Html dokumanının head etiketleri içerisinde uygulanır.

  • External CSS

    Html dokumanın dışında bir css dosyası oluşturularak sayfaya dahil edilerek uygulanır.

  • Inline CSS

    Html etiketlerinin içerisinde style attribute'u kullanılarak uygulanır.
