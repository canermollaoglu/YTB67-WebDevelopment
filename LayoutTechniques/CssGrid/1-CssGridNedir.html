CSS GRID LAYOUT

  • Grid Düzeni Modülü, satır ve sutunlardan oluşan bir ızgara tabanlı düzen sistemi sunar.
  • Grid Layout geliştiricilerin karmaşık web düzenlerini kolayca oluşturmalarına olanak tanır.
  • Float veya Position kullanmadan duyarlı bir düzen yapısını tasarlamayı kolaylaştırır.
  • Css Grid Layout özellikleri tüm modern tarayıcılar tarafından desteklenmektedir.

Grid vs Flexbox

  • Css Flexbox tek boyutlu düzenlemeye izin verir, sadece satır yada sadece sutun olarak ilerler.
  • CSS Grid Layout ise satır ve sutun olacak şekilde iki boyutlu düzenlemeye izir verir.
